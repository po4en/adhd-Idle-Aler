# adhd-Idle-Aler
Idle Alert is a small Windows helper that keeps an eye on your idle time and gently reminds you to come back to work when you’ve been away for too long. You can choose between a simple beep sound or a spoken voice phrase, so the reminder is easy to notice but not too annoying.

## Features

- Configurable idle threshold in minutes and seconds.
- Optional repeat interval (set to 0 for a one‑time alert).
- Two alert types: classic beep or voice message with custom text.
- Simple Start/Stop controls and live countdown to the next alert.
- “Buy me a coffee” button to support the project
  
## Usage

- Download Idle-alert.exe from the repository and run it.
- Set the idle threshold and repeat interval, choose alert type (Beep or Voice) and, for Voice, enter the phrase you want it to say.
- Click Start to begin monitoring; click Stop to stop the alerts.
